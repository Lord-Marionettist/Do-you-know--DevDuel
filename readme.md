## Tech Stacks (The contraints given)
* **HTML5:** Main Body.
* **CSS3:** Styling and layout.
* **JavaScript:** Game logic and interactivity.

## Features
* **Interactive Trivia:** Wide Range of Questions.
* **Responsive Design:** Works on every screen.
* **Instant Feedback:** You get ur score right after you answer the 10th question.
* **immersion:** An instresting loading screen and Background Music


## How to run

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Limitless-Demon-Venerable/Do-you-know--DevDuel.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Do-you-know--DevDuel.git
    ```
3.  **Launch the application:**
    Open `index.html` in your preferred web browser.
